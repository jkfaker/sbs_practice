#ss-download

### 下载保存图片，文档，视频到手机，兼容H5端，微信小程序端，APP端，	带进度条
 
### 不清楚使用方式可点击右侧导入示例项目运行完整示例

## 使用说明
| 属性		| 是否必填	|  值类型	| 默认值	| 说明			|
| --------- | -------- 	| -----: 	| --: 	| :------------:|
| fileUrl	|	是		| String	|空	    | 预览单个文档或视频传递url		|
| fileType	|	是 		| String	|空	    | 类型（1.预览图片，2.预览文件，3.预览视频）		|
